Notes for Review

  • · All screens are standalone HTML — no build step, no server required.
  • · Dark gradient nav + light content area — Fiona is an ops coordinator on a desktop, not a homeowner on a phone.
  • · Fuchsia accent throughout — #D946EF for primary actions, amber for warnings, emerald for approvals.
  • · Emotional Anchor pattern appears on all project-scoped screens — buyer motivation + communication tone guidance in a fuchsia-bordered card.
  • · "High" badge = complex JS interaction state (dashboard, project detail, messages).
  • · concierge_dashboard.html establishes all shared chrome — nav, sidebar, KPI pattern, table styles. Other screens inherit this layout.
